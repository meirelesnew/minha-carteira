# 🤖 Como usar este repositório com o Claude

Este arquivo explica como funciona nossa parceria de acompanhamento financeiro.

---

## Como atualizar todo mês

1. Abra o `README.md`
2. Atualize as tabelas com os dados do mês:
   - Quantas cotas de MXRF11 você comprou
   - Quanto depositou na caixinha do carro
   - Se teve freela: quanto ganhou
3. Salve e faça commit no GitHub
4. Cole o conteúdo do README no chat com o Claude e pergunte: **"Me orienta no próximo passo"**

---

## O que perguntar ao Claude

Cole sempre o README atualizado e use uma dessas perguntas:

- *"Analisando minha carteira, qual o próximo passo?"*
- *"Devo comprar mais MXRF11 ou focar na caixinha este mês?"*
- *"Tive um freela de R$ X. Como distribuo?"*
- *"A caixinha chegou em R$ 800. O que faço agora?"*
- *"Recebi dividendos de R$ X. Reinvisto tudo?"*

---

## Regras de ouro

| Situação | Ação |
|----------|------|
| Caixinha < R$ 800 | R$ 50 caixinha + R$ 50 MXRF11 |
| Caixinha = R$ 800 | R$ 100 inteiros em MXRF11 |
| Freela extra | 100% na caixinha até fechar R$ 800 |
| MXRF11 = 100 cotas | Perguntar ao Claude o próximo ativo |
| Dividendos recebidos | Reinvestir em MXRF11 até bater 100 cotas |
